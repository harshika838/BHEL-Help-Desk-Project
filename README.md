# AssetFlow — Smart Asset & Help Desk Management System

Full-stack IT asset tracking and help desk system, built during Summer Industrial Training at BHEL Haridwar.

**Roles:** Employee, IT Engineer, Admin — each with their own dashboard.

**Tech:** HTML/CSS/JS frontend, Node.js/Express/MySQL backend, JWT authentication.

## Built by
- **Aditya Gahlot** — Backend (API, database, auth)
- **Harshika** — Frontend (UI, integration with backend, added ticket-assignment feature)

## Run it
Backend: `cd backend`, `npm install`, set up `.env` (see `.env.example`), `node server.js`
Frontend: open any HTML file in `frontend/` with Live Server
